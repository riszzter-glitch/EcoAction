## ğŸš€ Cara Menjalankan Website di Localhost

### **Metode 1: Menggunakan Server Otomatis (TERMUDAH)**

1. **Double-click** file `START-SERVER.bat`
2. Browser akan terbuka otomatis di `http://localhost:8080`
3. Website Anda sudah berjalan!
4. Tekan `Ctrl+C` di terminal untuk menghentikan server

### **Metode 2: Menggunakan Python (Jika sudah terinstall)**

```bash
# Buka terminal di folder ini, lalu jalankan:
python -m http.server 8080

# Atau jika menggunakan Python 2:
python -m SimpleHTTPServer 8080
```

Kemudian buka browser di: `http://localhost:8080`

### **Metode 3: Menggunakan Node.js (Jika sudah terinstall)**

```bash
# Install http-server secara global
npm install -g http-server

# Jalankan server
http-server -p 8080
```

Kemudian buka browser di: `http://localhost:8080`

### **Metode 4: Menggunakan VS Code Live Server**

1. Install extension "Live Server" di VS Code
2. Klik kanan pada `index.html`
3. Pilih "Open with Live Server"
